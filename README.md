# VideoHub App

## Tech
- Backend: .NET 9, Dapper, JWT, Google OAuth, Stripe, Mux
- Frontend: Nuxt 3, TS, Tailwind+DaisyUI
- DB: SQL Server (Docker + Dapper)

## Setup
1. Clone repo.
2. Unesi env vars (MUX, STRIPE).
3. `docker-compose up --build`
4. Browse frontend http://localhost:3000
